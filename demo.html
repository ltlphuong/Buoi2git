Demo các câu lện GIT

GIT

  1. git clone url

    Lấy source code về local máy

  2. git init

    Lệnh tạo thư mục mới .git để lưu lại các thao tác

  3. git add tenfile

    Sau khi thay đổi phải cập nhật

  4. git commit -m tencongviecmuonluu

    Lưu lại sự thay đổi

  5. git config --global user.email "lethilanphuong17@gmail.com"

    git config --global user.name "ltlphuong"

    cung cấp danh tính (thường sd lần đầu git bash)

  6. git status

    kiểm tra trạng thái

  7. git remote add origin urlremote

    chưa tồn tại remote trên server thì phải add mới 1 remote rồi mới push

  8. git push origin master(tên git local)

    Cập nhật thông tin lên server

  9. git pull origin master

    Gộp những thay đổi của máy khác về local
